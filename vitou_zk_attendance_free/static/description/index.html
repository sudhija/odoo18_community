ZK Fingerprint Integration with Odoo

This module is used to integrate from odoo with zk-tico fingerprint device to download all staff attendance from device to odoo system.

Key Highlights

Device Setup and Integration

Can support ZK fingerprint device, with multi device, can use one device or can set one for ckeckin and other for checkout.

Dowonload Attendance

Can download all staff attendance from fingerprint device,

Advancetage

1- Download Attendance from device

2- Generate Work Hour base on checkin and checkout automatically without rouster.

3- Copy to odoo buildin module attandance.

4- Print Monthly Attendance by deparment list from day 1rst to end of Month.

Manual how to setup and use.

  1. In Python you need to install: attrgetter, pytz, zk, numpys
  2. - in cmd, Cd to: C:\Program Files\Python312
  3. - Pip3.12 install zklib
  4. - pip install numpy
  5. - python -m pip install Pyzk --upgrade
  6. Extact Zip File
  7. Copy and Past in folder custom_addons
  8. Restart Odoo Service
  9. Login Odoo as system administrator
  10. Navigate to Apps
  11. Update Apps List
  12. Search as keyword vitou or ZK-Biometric Device Integration
  13. Click Activate
  14. Then see the user manual bellow
